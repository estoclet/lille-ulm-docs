# Page spec - Contact

## Objectif de la page

Permettre au prospect de joindre Lille ULM rapidement, en confiance, sans friction inutile.

## Public cible

- prospect ayant une question avant achat ;
- client voulant un renseignement pratique ;
- visiteur preferant le telephone au paiement immediat.

## Blocs autorises

- intro courte rassurante ;
- telephone tres visible ;
- email ;
- formulaire court ;
- informations pratiques ;
- acces / localisation ;
- rappel sur delai de reponse.

## CTA principaux

- appeler ;
- envoyer le formulaire ;
- ecrire par email.

## Champs modifiables par le gestionnaire

- texte d'introduction ;
- telephone ;
- email ;
- horaires ou indications pratiques ;
- message de delai de reponse ;
- consentement / mentions du formulaire.

## Contraintes SEO

- titre explicite de type contact Lille ULM ;
- metadata dediees ;
- pas de titre generique "contact".

## Contraintes accessibilite

- labels de champs explicites ;
- messages d'erreur comprehensibles ;
- ordre logique de tabulation ;
- alternatives claires au captcha intrusif.

## Variantes autorisees

- formulaire avec ou sans choix de sujet ;
- carte affichee ou remplacee par des informations d'acces textuelles.

## Composants relies

- bloc coordonnees ;
- formulaire de contact ;
- bloc consentement ;
- bloc acces / localisation.
