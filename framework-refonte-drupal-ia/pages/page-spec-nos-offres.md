# Page spec - Nos offres

## Objectif de la page

Faire de `Nos offres` la page commerciale centrale du site et le point d'entree naturel vers l'achat.

## Public cible

- personne qui veut acheter un bapteme ;
- proche qui veut offrir un bon cadeau ;
- prospect qui compare durees et budgets.

## Blocs autorises

- hero avec promesse claire ;
- grille d'offres ;
- choix clair entre achat pour soi et achat a offrir ;
- bloc "ce qui est inclus" ;
- rassurance paiement / securite ;
- FAQ courte ;
- bloc contact humain ;
- CTA final.

## CTA principaux

- acheter cette offre ;
- offrir cette prestation ;
- nous appeler pour etre conseille.

## Champs modifiables par le gestionnaire

- titres ;
- textes courts ;
- prix ;
- disponibilite en carte cadeau ;
- eligibilite cadeau par offre ;
- ordre des offres ;
- visuels ;
- CTA ;
- messages de reassurance.

## Contraintes SEO

- titre de page explicite sur baptêmes ULM / Bondues ;
- H1 unique ;
- metadata specifiques a la page ;
- URL propre et stable.

## Contraintes accessibilite

- structure de titres coherente ;
- contrastes suffisants ;
- CTA explicites ;
- tableaux ou comparatifs evitables si une grille simple suffit.

## Variantes autorisees

- mise en avant d'une offre phare ;
- mise en avant d'une offre "a offrir" ;
- ordre des offres ajustable ;
- bloc FAQ affichable ou masquable.

## Composants relies

- cartes d'offre ;
- bloc choix "pour soi / pour offrir" ;
- bloc de reassurance ;
- bloc FAQ ;
- bloc contact rapide ;
- eventuel composant d'ajout au panier ou d'acces au parcours d'achat.
